# Android-Uitls
开发学习过程中 使用的工具类 提高开发效率
##类说明
- SPUtils
    - 封装xml处理
- StreamTools
    - 流转字符串
- HttpTools
    - 下载文件
    - 获取远程输入流
- CommonUtils 
    - 获取屏幕高宽、检测网络、px转dp、获取版本号和版本次数、通过Service的类名来判断是否启动某个服务
- Md5Utils
    - 通过Md5加密算法工具类
    
##使用方法
  下载根目录下的utils2.jar包
  在AndroidStudio中导入utils2.jar包即可
